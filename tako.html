📦 tako.js Documentation

A lightweight utility library built by Ayaan for fun and productivity.

🚀 Getting Started

Include tako.js in your HTML file:

<script src="https://ayaan-creator-web.github.io/Libraries/tako.js"></script>

🛠️ Available Functions

tako.isEmail(email)

Returns true if the input is a valid email address.

tako.isEmail("hi@example.com") // true

tako.isPhoneNumber(phone)

Returns true if the input is a valid international-style phone number.

tako.isPhoneNumber("+61412345678") // true

tako.pick(array)

Returns a random element from an array.

tako.pick(["apple", "banana", "pear"]) // "banana"

tako.capitalize(str)

Capitalizes the first letter of a string.

tako.capitalize("ayaan") // "Ayaan"

tako.kebabCase(str)

Converts spaces in a string to dashes (kebab-case).

tako.kebabCase("hello world") // "hello-world"

tako.parity(number)

Returns "Even" or "Odd" based on the number.

tako.parity(42) // "Even"

tako.convert(value)

Converts a string to a number or a number to a string.

tako.convert("123") // 123
tako.convert(456) // "456"

tako.isURL(url)

Returns true if the input is a valid URL.

tako.isURL("https://example.com") // true
